Instructions

Firstly, read both of the "How It Works" pages, "How 8 Bit CPUs Work" and "How Do CPUs Work". After reading this pages you can go to either the "Live View" page if you just want to watch others input commands or to the "CPU Programming" page if you want to input commands. Once there make sure to input commands in a logical sense and only set binary values in the value box up to 1111. Once input a command, watch as your command sequence displays on the right, NOP displaying for all of the locations not used. Watch as the stream displays the CPUs response to your commands and enjoy!
